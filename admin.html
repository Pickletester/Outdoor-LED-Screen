Filename schedule reference
__sYYYYMMDD — start date (inclusive)
__eYYYYMMDD — end date (exclusive)
__dSECONDS — per-image duration override
__off — disabled; ad stays in folder but never shows
__wMON,FRI3 — weekday rule; e.g. every Mon + 3rd Fri. Letters: MON/TUE/WED/THU/FRI/SAT/SUN. Add 1–5 for Nth week, or L for last week.
